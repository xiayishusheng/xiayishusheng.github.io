Ubuntu

Ubuntu设置不睡眠

16.04以下的旧版本
打开 System Settings –> Power（中文版是打开 系统设置 -> 电源），选择不挂起

18.04版本
编辑 Login Manager 的配置文件（logind.conf）
sudo vim /etc/systemd/logind.conf
打开文件后修改下面这行：
#HandleLidSwitch=suspend
改成这样：
HandleLidSwitch=ignore
保存文件，重启 Login Manager 服务：
sudo service systemd-logind restart
or
systemctl restart systemd-logind

选项值的意义：

  • ignore(无操作，不过合上笔记本屏幕会熄灭)
  • supend(休眠到内存)
  • hibernate(休眠到硬盘)
  • poweroff(关闭系统并切断电源)
  • halt(关闭系统但不切断电源)
  • hybrid-sleep(同时休眠到内存和硬盘)
  • lock(锁屏)
